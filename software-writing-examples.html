Import APAX-5000/6000 Variables

WebAccess supports the importing of Advantech APAX-5000 and 6000 series variables, including Tag names and Tag addresses, from the KW CSV file to the WebAccess database. It automatically converts the KW variable address to a Modbus address for runtime exchange real-time data.

Step 1: KW CSV Configuration
When you make all the IO configurations and variable settings, execute the following instructions to export all the variables (Global Variables only):

1.Select File and Export: Export the I/O information to the CSV file from the command line.



2.Select Cross reference from the pop-up dialog box and click OK.



3.Save CSV file: Keep the default file type-CSV - and specify the file name and path.

When a customer orders a product which is too large to go into a regular container it's classed as a **bulky item**. For the purposes of picking bulky items are treated as **containers** and are given their own barcode.

{: .note}
> Bulky items are only suitable for **Customer Collection** and only work if in-store fulfillment is configured to use **SKUs**.
>
> Bulky item SKUs must be marked as **Bulky** using OSP's SKU API.
## Manage equipment

Before you can start picking bulky items you need to add a bulky items picking cart.

1. From the **Welcome** screen click ☰.

2. Click **Manage equipment**.

3. Click **New bulky items picking cart**.

4. Select an item from the list and click **NEXT**.

5. Scan the picking cart's barcode anc click **NEXT**.

6. Check the summary and click **CONFIRM**.

7. You can now either **ADD MORE** bulky item picking carts or click **DONE**.

## Pick bulky items

{: .note }
> All temperature regimes are picked at the same time.

Before you start picking bulky items ensure that you have enough printed barcodes for the items you're picking. These can be printed from the [**Store operations dashboard**](../../store-pick/dashboard#pbl).

The picking process is as follows:

1. Scan the product's barcode.

2. Stick the picking barcode on to the product.

3. Scan the picking barcode.
